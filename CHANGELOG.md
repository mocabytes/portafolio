# Changelog

## v1.0.0 — Primera versión estable

### Resumen
- Lanzamiento de la primera versión estable del portafolio: interfaz finalizada y despliegue en producción.

### Novedades destacadas
- Diseño responsivo completo y mejoras de accesibilidad.
- Secciones: Sobre, Proyectos, Blog y Contacto implementadas.
- Integración con el sitio: https://www.mariaserrano.tech/
- Optimización de assets y tiempos de carga.

### Correcciones
- Correcciones menores en estilos y layout en móviles.
- Mejor manejo de enlaces externos.

### Cambios incompatibles / migración
- Ninguno. Es la primera versión estable.

### Cómo usar / desplegar
- Repositorio en TypeScript, dependencias en package.json.
- Para ejecutar localmente: `npm install && npm run dev`
- Para construir: `npm run build`
