Welcome to the repository for my personal developer portfolio! This project is crafted to showcase my technical skills, professional background, and recent software development work.

## 🚀 Overview

Built using **Astro** for exceptional static site performance and **React** for dynamic, interactive components, this portfolio provides a seamless user experience. It highlights my journey in software engineering, from UI/UX implementation to complex backend architectures.

### ✨ Key Features
* **Lightning Fast:** Leverages Astro's Island architecture to send minimal JavaScript to the client.
* **Component-Driven:** Built with reusable React components (`ProjectCard`, `TechCardGrid`, `AnimatedCounter`, etc.).
* **Responsive Design:** Fully fluid layout using modern CSS, ensuring a great experience on mobile devices and desktop screens.
* **Project Showcase:** Includes dedicated sections and a custom lightbox (`ProjectLightbox`) to display application screenshots (e.g., the Medu digital inventory system).
* **Interactive Elements:** Features social circles, status icons, and dark veil backgrounds for a polished visual aesthetic.

## 🛠️ Tech Stack & Architecture

* **Framework:** Astro
* **UI Library:** React (TSX)
* **Language:** TypeScript
* **Styling:** Custom Modular CSS (`animations.css`, `tokens.css`, `global.css`)
* **Assets:** Optimized SVG and WebP imagery for high performance.

## 📂 Project Structure

A quick overview of the file structure:

```text
/
├── public/                 # Static assets (images, icons, resume PDF)
│   ├── images/medu/        # Screenshots for Project Medu
│   └── Maria_Desarrolladora_FullStack.pdf
├── src/
│   ├── components/         # Reusable UI components
│   │   ├── react/          # React-specific components (Forms, Lightbox, Grids)
│   │   ├── sections/       # Astro components for page sections (Hero, Profile, Projects)
│   │   └── ui/             # Core UI elements
│   ├── data/               # Content data (tech stack, projects, education, strengths)
│   ├── layouts/            # Page layout wrappers (BaseLayout)
│   ├── pages/              # Application routes (index.astro, API endpoints)
│   └── styles/             # Global CSS, design tokens, and animations
├── astro.config.mjs        # Astro configuration file
└── package.json            # Project dependencies and scripts
```
